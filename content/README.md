# Content

Content registries hold starter command, mode, panel, artifact, evolution, and companion definitions.

The initial split is simple:

- `commands/`
- `modes/`
- `panels/`
- `artifacts/`
- `ambient/`
- `micromotion/`
- `distortion/`
- `capabilities/`
- `evolution/`
- `companions/`

These files give the first implementation a stable data source before more dynamic authoring tools exist.
